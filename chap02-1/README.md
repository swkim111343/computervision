## 컴퓨터비전 챕터2-1 과제
1. OpenCV를 이용하여 개발된 제품을 조사하고 기능을 설명하라.<span style="color:red">빨간 글씨</span>
    - 자율주행 자동차의 객체 인식 시스템으로, 자율주행 차량이나, 운전자 보조 시스템(ADAS)이 있다.
    - OpenCV를 이용하여 차량 주변 환경, 보행자, 차량, 신호등, 차선 등을 인식한다.
    - 차량 카메라로 영상을 입력하고 OpenCV로 영상 전처리 후 객체 탐지 알고리즘으로 물체를 탐지하고 차량제어 시스템에 전달한다.

2. 소프트웨어 분야의 용어인 라이브러리의 의미를 자세히 설명하라.
    - 프로그램을 개발할 때 자주 사용하는 함수, 클래스, 데이터 등을 미리 만들어 모아 놓은 코드 집합을 의미한다.
    - 개발자는 라이브러리를 재사용할 수 있기 때문에 이를 통해 프로그램 개발시간을 줄이고 효율적인 개발을 할 수 있다.

3. 오픈소스의 의미를 설명하라.
    - 말그대로 열려있는 소스코드로 누구나 자유롭게 열람하고 사용할 수 있도록 공개한 소프트웨어 개발방식이다.

4. Windows 운영체제의 환경변수 중 Path 변수의 기능을 설명하라.
    - 운영체제가 실행 파일이나 프로그램의 위치를 찾을때 참조하는 디렉터리 경로들을 저장하는 변수이다.

5. 프로그램 개발과정인 편집, 컴파일, 링크, 실행 단계를 각각 설명하라. 특히, 링크단계에서 라이브러리의 역할에 대하여 설명하라.
    - 편집 : 프로그래밍언어를 사용하여 소스 코드 작성, 수정 단계
    - 컴파일 : 사용자가 작성한 소스코드(고급언어)를 기계어(저급언어)로 변환 하는 단계
    - 링크 : 목적파일(obj)과 라이브러리를 결합하여 하나의 실행파일을 만드는과정.
    - *(여기서 라이브러리의 역할은 외부함수제공, 목적 파일과 라이브러리 결합하여 하나의 실행 파일 생성, 실행가능한 프로그램 완성의 역할이 있다.)*
    - 실행 : 실행 파일을 실제로 실행하여 프로그램 동작과정 단계

6. 정적 라이브러리(SLL)와 동적 라이브러리(DLL)에 대하여 조사하라.
    - SSL : 프로그램을 컴파일하고 링크하는 과정에서 실행 파일에 직접포함 되는 라이브러리이다.(프로그램에 라이브러리 코드가 직접 포함됨)
    - DLL : 프로그램 실행 시 운영체제가 필요한 라이브러리를 메모리에 불러와 연결하는 방식의 라이브러리이다.(실행 파일과 라이브러리가 분리되어 있음)

7. 동적 라이브러리(DLL)의 경로는 운영체제의 환경변수 PATH에등록되어야 한다. 이유를 설명하라.
    - 프로그램 실행 시 운영체제가 DLL 파일의 위치를 찾아야 하는데, PATH에 등록되어 있으면 해당 경로를 자동으로 검색할 수 있다. 즉 쉽게 찾기 위해 등록해야한다.
