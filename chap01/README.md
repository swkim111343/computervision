## 컴퓨터비전 챕터1 과제
1. 컴퓨터 비전기술이 적용된 제품을 찾아 제품의 기능, 동작원리 등을
조사하여 제출하시오.
    - 스마트폰 얼굴 인식(Face ID)
    - 카메라 + IR 센서가 얼굴을 촬영
    - 얼굴 특징점(landmarks) 추출
    - 3D 얼굴 모델 생성
    - 저장된 사용자 모델과 대조하여 매칭

2. 영상처리와 컴퓨터 비전의 차이점을 자세히 설명하시오.
    - 영상처리(Image Processing) : 영상을 변형하거나 개선하는 기술 
    ex) 밝기 조정, 필터 적용, 노이즈 제거
    - 컴퓨터 비전(Computer Vision): 영상에서 의미를 이해하고 해석하는 기술
    ex) 객체/사람 인식, 분류, 행동 분석

3. CMOS 센서의 원리를 자세히 조사하시오.
    - 각 픽셀이 빛을 받으면 전하가 생성되어 받은 빛의 양에 비례해서 전하량이 결정됨, 생성된 전하는 트랜지스터를 통해 전압으로 변환(ADC)아날로그 투 디지털 변환기로 디지털 값으로 변환됨

4. 영상의 해상도와 이미지 센서의 개수의 관계를 설명하시오.
    - 해상도가 높아질 수록 픽셀 수가 많아지고, 이는 이미지 센서의 수도 많아진다는 뜻

5. 100X100 그레이스케일 영상 파일의 크기(바이트)를 계산하라.
    - 픽셀수 : 100X100=10,000 
    - 1픽셀은 8비트 이므로 값:0~255
    - 총 파일 크기는 10,000X1=10,000바이트

6. 100X100 컬러 영상 파일의 크기(바이트)를 계산하라
    - 컬러영상은 보통 RGB 3채널 -> 10,000X3=30,000바이트
